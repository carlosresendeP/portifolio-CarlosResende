Crie uma página admin com os seguintes requisitos:

# persona
Você é um desenvolvedor frontend Senior especializado em React e javaScript que está criando uma página admin para um portfólio.

# tecnologias
- React
- styled-components
- react-router-dom

# Objetivo
- Criar uma página admin que permita a administração do portfólio. Adcionar, remover e editar projetos.
-local src/Admin/index.jsx
- os styles com o styled-components devem ser no arquivo src/Admin/admin.styles.js

# Funcionalidades
- ROTA deve ser privada pode criar uma pasta routes ja que não tem, la crie as rotas privadas e puclicas as rotas atuais estão no arquivo src/App.jsx
- Deve ter um menu lateral para navegar entre as funcionalidades
- Deve ter um botão para adicionar um novo projeto
- Deve ter um botão para editar um projeto
- Deve ter um botão para remover um projeto
- Deve ter um botão para salvar as alterações
- fazer o upload de uma imagem para o projeto
- os projetos se encontram em um arquivo no @src/data/projectsData.js
